# COP v1 — cTrader Order Panel

Торговый робот для cTrader с графической панелью для быстрого размещения лимитных и рыночных ордеров. Объём позиции рассчитывается автоматически на основе заданного процента риска от баланса.

## Возможности

- **Limit / Market** — переключение между лимитным и рыночным ордером.
- **Три линии на графике** — Entry, Stop Loss, Take Profit (перетаскивание или ввод в полях).
- **Расчёт объёма по риску** — введите макс. риск в % — бот рассчитает лоты по расстоянию до SL.
- **Быстрый ордер (Fast Order)** — фиксация уровней кликами мыши: Limit (Entry → SL → TP), Market (SL → TP).
- **Локализация** — интерфейс на английском и русском.
- **Сворачиваемая панель** — состояние сохраняется при смене таймфрейма (по умолчанию свёрнута).

## Требования

- cTrader (cAlgo).
- .NET 6.0 (указывается в проекте).

## Установка

1. Клонируйте репозиторий или скопируйте папку `cop v1` в каталог роботов cAlgo (например `Documents/cAlgo/Sources/Robots/`).
2. Откройте решение `cop v1.sln` в Visual Studio или добавьте робота в cTrader.
3. Соберите проект и запустите робота на нужном графике.

## Параметры робота

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| **Vertical Position** | Вертикальное положение панели (Top / Center / Bottom) | Top |
| **Horizontal Position** | Горизонтальное положение (Left / Center / Right) | Right |
| **Max Risk %** | Максимальный риск в % от баланса для расчёта объёма | 2.0 |
| **Fast Order Mode** | Включить режим быстрого ордера по умолчанию | false |
| **Interface Language** | Язык интерфейса (EN / RU) | EN |

## Использование

1. Запустите робота на графике — появится панель (по умолчанию свёрнута; нажмите ▼ чтобы развернуть).
2. При необходимости включите **Spread** и/или **Fast order**.
3. Выберите **Limit** или **Market** — на графике появятся линии.
4. Перетащите линии или введите цены в полях. Объём, убыток, прибыль и RR пересчитываются автоматически.
5. Нажмите зелёную кнопку подтверждения (например «BUY EURUSD 0.01 лот») для размещения ордера.

В режиме **Fast order**: для Limit — три клика (Entry, SL, TP); для Market — два клика (SL, TP). Ордер выставляется автоматически после последнего клика.
