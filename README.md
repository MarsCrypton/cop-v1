# COP v1 — cTrader Order Panel

Торговый робот для cTrader с интерактивной панелью на графике: лимитные и рыночные ордера, до **трёх тейк-профитов**, расчёт объёма из заданного риска (% депозита, USD или EUR), перетаскиваемые линии Entry / SL / TP и режим **Fast Order** (фиксация уровней кликами).

## Возможности

- **Limit / Market** — переключение режима; в Market цена входа привязана к текущей котировке.
- **До 3 тейк-профитов** — в настройках «…» выбирается число тейков (1–3) и **режим распределения объёма**: равные лоты на каждый TP или **равный денежный профит** на тейк (в валюте счёта). При нескольких TP бот размещает **несколько ордеров** с общим SL и разными TP (доли объёма считаются в `RiskCalculator`).
- **Три линии на графике** (и дополнительные линии TP при multi-TP) — перетаскивание или ввод цен в полях; подписи на линиях (в т.ч. режимы **LM** / **MK** для Limit/Market) с якорем по времени, чтобы при прокрутке графика текст не «лип» к краю.
- **Риск и объём** — поле риска на панели с переключением режима: **процент баланса**, фиксированная сумма **USD** или **EUR** (конвертация в валюту счёта через API). Объём нормализуется по шагу символа и ограничениям брокера.
- **Спред** — всегда отображается на панели (в пипсах), без отдельного чекбокса.
- **Fast Order** — клики по графику: для **Limit** задаются Entry → SL → TP1 → (TP2) → (TP3); для **Market** — SL → TP1 → … После последнего клика ордер(а) отправляются автоматически (см. логику в `FastOrderHandler`).
- **Панель настроек «…»** — открывается под основной панелью, та же ширина/стиль. **Вкладки** (состояние активной вкладки сохраняется): **Торговля** (число тейков, режим объёма), **Панель** (прозрачность фона 0–80 %, масштаб UI 80–150 %), **Ещё** — заглушка под будущие опции.
- **Масштаб и прозрачность** — задаются в параметрах робота **и** в панели; значения синхронизируются с `LocalStorage` (приоритет: сохранённое значение vs изменение параметра — см. `COP.OnStart`).
- **Локализация** — интерфейс панели на **9 языках**: EN, RU, DE, FR, ES, IT, PL, NL, PT (`Localization.cs`). Параметры экземпляра робота в cTrader по-прежнему **только на английском**.
- **Свёрнутая «мини-панель»** — компактный заголовок с ▼/▲; подсказки/статус (в т.ч. плейсхолдер под подписку в свёрнутом виде). Состояние свёрнутости и черновики уровней сохраняются.
- **Смена таймфрейма** — объекты графика уничтожаются платформой; `ChartLineManager` кэширует цены и при следующем тике восстанавливает линии (`RepairTradingLinesIfNeeded`), уровни также чернятся в `LocalStorage` при остановке робота.

## Требования

- cTrader / cAlgo.
- **.NET 6.0** (как в `cop v1.csproj`).

## Сборка

```bash
dotnet build "cop v1/cop v1.csproj"
```

Артефакт: `cop v1/bin/Debug/net6.0/cop v1.algo` — подключите в AlgoTrader / добавьте робота на график.

Автотестов в репозитории нет; проверка — в симуляторе или бэктесте cTrader.

## Установка

1. Склонируйте репозиторий или скопируйте папку `cop v1` в каталог роботов cAlgo (например `Documents/cAlgo/Sources/Robots/`).
2. Откройте `cop v1.sln` в Visual Studio или добавьте проект в cTrader.
3. Соберите проект и запустите робота на нужном графике.

## Параметры робота

Названия групп, параметров и значений перечислений в окне настроек cAlgo — **на английском** (требование платформы).

| Параметр | Группа | Описание | По умолчанию |
|----------|--------|----------|--------------|
| **Vertical Position** | Panel alignment | Вертикальное положение панели (Top / Center / Bottom) | Top |
| **Horizontal Position** | Panel alignment | Горизонтальное положение (Left / Center / Right) | Right |
| **Panel transparency %** | Panel alignment | Прозрачность фона панелей, 0–80 | 0 |
| **Panel scale %** | Panel alignment | Масштаб UI: Scale80 … Scale150 (шаг 10 %) | Scale100 |
| **Risk Mode** | Default trade parameters | Режим поля риска на панели: Percent / USD / EUR | Percent |
| **Max Risk %** | Default trade parameters | Макс. риск в % баланса (0.1–100) | 2.0 |
| **Max Risk USD** | Default trade parameters | Макс. риск в USD | 50.0 |
| **Max Risk EUR** | Default trade parameters | Макс. риск в EUR | 50.0 |
| **Fast Order Mode** | Default trade parameters | Включить Fast Order при старте (No / Yes) | No |
| **Interface Language** | Default trade parameters | Язык текста панели (EN … PT) | EN |

## Использование

1. Запустите робота на графике. Панель по умолчанию может быть **свёрнута** — нажмите ▼, чтобы развернуть.
2. При необходимости откройте **«…»** и настройте число тейков, режим объёма, прозрачность и масштаб.
3. Включите **Fast order** (тумблер OFF/ON), если нужен ввод уровней кликами.
4. Выберите **Limit** или **Market** — появятся линии (и дополнительные TP при multi-TP).
5. Перетащите линии или введите цены. Отображаются пересчёт объёма, оценка P/L, RR; для нескольких TP — показатели с учётом выбранного режима распределения.
6. Нажмите кнопку подтверждения сделки (например покупка/продажа с объёмом) — для multi-TP будут отправлены **несколько** ордеров с разными TP.

**Fast Order:** Limit — последовательность Entry, SL, затем TP по количеству в настройках; Market — SL, затем TP. После финального клика срабатывает отправка.

## Сохранение состояния

Через `LocalStorage` (Device) сохраняются, в числе прочего: режим и значения риска, свёрнутость панели, прозрачность, масштаб, активная вкладка настроек, параметры multi-TP, **черновик цен** линий для восстановления после смены таймфрейма или перезапуска (см. `COP.cs`, `ChartLineManager`).
