# Updates Flow

Единый поток внедрения обновлений в проекте COP v1.

## Этапы

| Этап | Папка | Назначение |
|---|---|---|
| 1. Planned | `docs/updates/future-updates` | Идеи и будущие изменения, которые еще не взяты в работу |
| 2. In Progress | `docs/updates/in-development` | Обновления, которые сейчас реализуются |
| 3. Done | `docs/updates/archive` | Завершенные и внедренные обновления |

## Правило синхронизации с Linear

- `Backlog/Todo` -> `docs/updates/future-updates`
- `In Progress` -> `docs/updates/in-development`
- `Done` -> `docs/updates/archive`

## Связь с TODO

Основной список задач ведется в [TODO.md](../../TODO.md).
При переводе задачи между этапами обновляйте и статус в Linear, и расположение/документ в соответствующей папке `docs/updates`.
