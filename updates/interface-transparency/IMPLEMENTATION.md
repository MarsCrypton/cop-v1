# Изменение прозрачности интерфейса

## Цель обновления

Сделать **основной фон трёх панелей** настраиваемо прозрачным:

- **Основная панель** (фон контейнера)
- **Мини‑панель** (при свёрнутой панели)
- **Панель настроек** (окно «...»)

По умолчанию — **10%** прозрачности. Диапазон: **0–80%**.

---

## Текущая структура (по коду)

В проекте используются три визуальные панели с единым фоном из `PanelStyles`:

| Панель | Контрол в коде | Назначение |
| ------ | -------------- | ---------- |
| Основная панель | `_rootBorder` (Border) | Один слой фона для основной части панели |
| Мини‑панель | `_miniPanelStack` (StackPanel) | Вторая строка при свёрнутой панели |
| Панель настроек | `_settingsPanelBorder` (Border) | Отдельная панель под основной |

---

## UX/Интерфейс: как должно выглядеть

### Параметр робота (Settings cTrader)

- **`Panel transparency %`**: `int`, диапазон **0–80**, по умолчанию **10**

### В панели (Settings «...»)

Добавить настройку:

- **Panel transparency**: `0% … 80%` шагом `10%`

При изменении значения:

- фон **основной панели** и **панели настроек** обновляется сразу
- значение сохраняется в `LocalStorage` и используется при следующем запуске

---

## Реализация (план)

1) **`PanelStyles.cs`**
- Добавить метод `GetPanelBackgroundWithTransparency(int transparencyPercent)`, который возвращает `Color.FromArgb(alpha, PanelBackground)`.

2) **`MainPanel.cs`**
- Принять параметр `panelTransparencyPercent` в конструкторе.
- Использовать `PanelStyles.GetPanelBackgroundWithTransparency()` для `BackgroundColor` у `_rootBorder` и `_settingsPanelBorder`.
- Добавить `ComboBox` в панель настроек и событие `OnTransparencyChanged(int percent)`.

3) **`COP.cs`**
- Добавить параметр робота `PanelTransparencyPercent`.
- Загружать сохранённое значение из `LocalStorage` (если есть) и передавать в `MainPanel`.
- На событие `OnTransparencyChanged` сохранять значение в `LocalStorage`.
