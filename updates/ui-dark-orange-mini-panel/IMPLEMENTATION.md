# Реализация: тёмно-оранжевый UI

## Сделано

- **Палитра и токены** — [`cop v1/UI/PanelStyles.cs`](../../cop%20v1/UI/PanelStyles.cs): фон `#0D0D0D`, акцент `#F25F00`, приглушённый текст `#666666`, полоса шапки `#1A1A1A`, линии `#1F1F1F`, рамка `#333333`, индикатор статуса `#00C853`, скругление панели 10 px.
- **Основная панель** — шапка с оранжевым маркером; заголовок переключается «COP v1» / локализованный мини-заголовок при свёртывании; разделители и внешняя рамка по новым токенам.
- **Свёрнутый режим** — вертикальный блок: ряд LM/MK/OK/FST, линия, футер («Свернутая панель» + зелёный индикатор + «Активна»).
- **Панель настроек** — шапка и лейблы строк в стиле макета (`HeaderBarColor`, `TextMuted`).
- **Локализация** — ключи `MiniPanelTitle`, `CollapsedPanelHint`, `StatusActive` для EN/RU/DE/FR/ES/IT/PL/NL/PT.

## Проверка

- `dotnet build` в каталоге проекта `cop v1` — успешно.
- Ручной прогон в терминале cTrader: разворот/сворачивание, настройки, прозрачность, кнопки Limit/Market/OK.
