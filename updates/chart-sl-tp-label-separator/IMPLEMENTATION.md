# Подписи SL / TP: разделитель между числом и суммой

## Зачем

На линиях графика нужен **одинаковый** визуальный разделитель между двумя числовыми частями:

- **Stop Loss:** `SL {процент}%` **·** `{сумма $}` — уже было в `StopText` (символ U+00B7 middle dot).
- **Take Profit:** `TP RR {число}` **·** `{сумма $}` — раньше между RR и долларами был только пробел.

Средняя точка **`·`** типографски ближе к **центру высоты строки**, чем обычная точка `.` у нижнего края.

## Изменения

- **`cop v1/UI/Localization.cs`** — ключ **`TpText`**: для всех языков шаблон `TP RR {0} · {1}` вместо `TP RR {0} {1}`.

Код вызова (`COP.RecalculateAll`, `FastOrderHandler`, плейсхолдеры `InitialTpLineLabel`) не менялся — те же два аргумента.

## Альтернатива

При желании тот же символ можно заменить на тире, например **en-dash** ` – `, в обоих ключах `StopText` и `TpText` единообразно.
